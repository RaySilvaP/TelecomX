# 📊 Análise de Evasão de Clientes (Churn) -- TelecomX
Desáfio TelecomX da Alura/Oracle One

## 📌 Sobre o Projeto

Este projeto tem como objetivo analisar a evasão de clientes (Churn) da
empresa fictícia **TelecomX**, identificando padrões e fatores que
influenciam o cancelamento de serviços.

A evasão de clientes é um dos principais desafios de empresas de
telecomunicações, pois impacta diretamente:

-   Receita recorrente\
-   Custos de aquisição de novos clientes\
-   Crescimento sustentável do negócio

Por meio de uma Análise Exploratória de Dados (EDA), o projeto busca
compreender o comportamento dos clientes e gerar insights estratégicos
para redução do churn.

------------------------------------------------------------------------

## 🎯 Objetivo

Identificar padrões associados ao cancelamento de clientes e transformar
os dados em recomendações estratégicas que ajudem a empresa a:

-   Reduzir a taxa de evasão\
-   Aumentar a retenção\
-   Melhorar a proposta de valor\
-   Criar estratégias segmentadas

------------------------------------------------------------------------

## 🗂 Base de Dados

O projeto utiliza o arquivo:

`TelecomX_Data.json`

A base contém informações como:

-   Dados demográficos\
-   Tipo de contrato\
-   Serviços contratados\
-   Tempo de permanência\
-   Valores cobrados\
-   Status de cancelamento (Churn)

------------------------------------------------------------------------

## 🔎 Spoiler dos Resultados

Alguns dos principais achados da análise:

-   📉 Clientes com **contrato mensal** apresentam taxa de churn
    significativamente maior.
-   ⏳ A evasão ocorre principalmente nos **primeiros meses de
    contrato**.
-   💰 Planos mais caros (como fibra óptica) possuem maior cancelamento,
    sugerindo possível problema de custo-benefício.
-   👨‍👩‍👧 Clientes com parceiro(a) e dependentes tendem a apresentar maior
    retenção.
-   🛡 Serviços de suporte e segurança ajudam a reduzir a evasão.

Esses resultados indicam que estratégias focadas em retenção inicial,
revisão de preços e segmentação de público podem gerar impacto
significativo.
